My Interests

Web Development

I started my web development journey long ago at my first Hackathon in highschool where I used HTML and CSS to create a static webpage. Looking back, I have come a long way and now have full-stack web development experience. Even after so many years it continues to excite me since it gives me the power to bring visions for applications into reality. On top of that the field forces you to learn and stay on top as it is always involving and introducing powerful frameworks.

Machine Learning

The idea behind Artificial Intelligence and its future applications is in my opinion one of the most captivating areas of computer science. This past year I've been spending a lot of time using Python and Tensorflow 2.0 to explore the power to machine learning and nueral networks. More specifically, I am very interested in Reinforcement Learning and Computer Vision and spend time on the side learning about model development.

Stock Trading

I began teaching myself about the stock and FOREX markets this summer and have learned a lot about technical analysis and swing trading. This field interests me since I love trying to make logical sense behind the movements of the stock market, and in the future want to explore automated trading systems and try to get an edge in the market using AI. As for now, I make some small trades based on my analysis and am also looking more into the Options market.

My Work

Past and Ongoing Projects

3D Lunar Lander Game

Used C++ and OpenFrameworks to create a fully functional 3D Lander-style game that incorporates full physics simulation for movement, octree subdivision for collision and altitude detection, shaders for visual effects, and proper lighting and controls for the user.

GitHub Repository
Cooked It

Fully designed and created a RESTful API for CookedIt's (a web-based service allowing users to share recipes) backend utilizing Node.js, Express, and MongoDB. Created CRUD routes for users, profiles, and posts. Created auth middleware and used JWT to verify user authentication.

GitHub Repository
Reddit Clone

I created Reddit Clone, a full-stack application, for Mintbean's 50th hackaton over one weekend. I used RedwoodJS, a framework that uses React, Prisma, and GraphQL to make this application, and desgined it using ANT Design. It features a posts wall and a subreddit listing page, and allows the user to create their own posts and subreddits.

GitHub Repository
Got Broke

Got Broke is a cluster-free and simple budgeting application made using purely JAVA. It helps users manage their income, savings, and spendings for a given month. It then displays these results in a summary page, making it convenient for them to easily see their monthly budget.
