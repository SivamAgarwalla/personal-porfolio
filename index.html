My Interests

Web Development

Machine Learning

My interest in this field of started at the beginning of this year when I sat in on the lectures of SJSU's machine learning class. Here I learned about the current applications and future potentials of machine learning, the mathematical foundations behind it, and how to design neural networks and tune different deep learning models. Over the summer I further reinforced my conceptual knowledge and learned more about different fields of deep learning. Right now, I’m currently working through a Tensorflow 2.0 course on Udemy to become more comfortable with this popular framework. I'm diving into RNNs and Reinforcement Learning and look forward to exploring more about them with SJSU's ML Club this semester.

Stock Trading

The possibility of generating side income from the stock market was always on my radar, and this summer I forced myself to sit down and learn about the complexities of the stock market, with a focus on technical analysis. I learned about all the terms, factors that push market movement, and reading candlestick charts and identifying common patterns. I focus on price action trading over the multi-day time frame and am currently developing and back testing trading strategies in my paper trading account in the NYSE, FOREX, and Options markets. Stocks keep me busy and serve as a constant reminder about life’s uncertainty. In the future, I aim to find a profitable trading plan and automate it!

My Work

Noteworthy Past Projects

3D Lunar Lander Game

Used C++ and OpenFrameworks to create a fully functional 3D Lander-style game that incorporates full physics simulation for movement, octree subdivision for collision and altitude detection, shaders for visual effects, and proper lighting and controls for the user.

GitHub Repository
Cooked It

Fully designed and created a RESTful API for CookedIt's (a web-based service allowing users to share recipes) backend utilizing Node.js, Express, and MongoDB. Created CRUD routes for users, profiles, and posts. Created auth middleware and used JWT to verify user authentication.

GitHub Repository
Reddit Clone

I created Reddit Clone, a full-stack application, for Mintbean's 50th hackaton over one weekend. I used RedwoodJS, a framework that uses React, Prisma, and GraphQL to make this application, and desgined it using ANT Design. It features a posts wall and a subreddit listing page, and allows the user to create their own posts and subreddits.

GitHub Repository
Got Broke

Got Broke is a cluster-free and simple budgeting application made using purely JAVA. It helps users manage their income, savings, and spendings for a given month. It then displays these results in a summary page, making it convenient for them to easily see their monthly budget.
